# Python Virtual Proving Ground

Среда моделирования для запуска и тестирования сценариев, а также (опционально) интеграции моделей машинного обучения.

---

## Структура проекта

python-virtual-proving-ground/
│
├── rl/ # Модели машинного обучения и связанные данные
│
├── scenarios/ # Описание и конфигурации сценариев
│
├── sim/ # Основная логика симуляции
│ ├── draw.py # Отрисовка и визуализация
│ ├── entities.py # Сущности (машины и т.д.)
│ ├── physics_engine.py # Физика симуляции
│ └── scenario_loader.py # Загрузка сценариев
│
├── tests/ # Unit-тесты
│ ├── test_entities.py
│ ├── test_physics_engine.py
│ └── test_scenario_runner.py
│
├── scenario_creator.py # Инструмент создания сценариев
├── scenario_runner.py # Запуск сценариев
│
├── config.ini # Конфигурация проекта
├── README.md # Документация


---

## Инструкция для начала работы

### 1. Требования

- Python 3.15  
- Библиотека `pygame`

Установка зависимостей:

```bash
pip install pygame
2. Запуск визуализации

Для запуска визуализатора сценариев выполните:

python scenario_runner_gui.py

По умолчанию будут запущены все сценарии из папки scenarios/.

## Описание компонентов
sim/

Содержит основную логику симуляции:
- расчёт физики
- управление сущностями
- отрисовка
- загрузка сценариев
- scenarios/

Хранит сценарии симуляции.
Каждый сценарий описывает начальные условия, объекты и параметры среды.

rl/
Предназначен для работы с моделями машинного обучения:

- хранение моделей
- результаты обучения

Unit-тесты для проверки корректности работы:
- физики
- логики сущностей
- выполнения сценариев
